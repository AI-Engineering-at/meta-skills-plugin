## 5. Neue Learnings diese Session

### C-CLAIM03 — Silent baseline-backfill wipe
Siehe `self-improving/corrections.md.example` Sektion "Session 2026-04-17/18".

**Kurz:** `~/.claude/statusline-alltime.json` verlor `baseline-backfill` 2× mitten in der pathlib-Migration. Root-Cause: alte `statusline.py` hat bei JSON-Decode-Error auf dem Stats-File die Datei überschrieben. Fix: Read-path hardening via `_stats_write_ok` flag (differentiate "missing" vs "corrupt"). Prevention: `tests/test_statusline_stats.py` (23 Tests).

### Lessons für nächste Session
1. **Branch-Awareness:** 2× passiert dass ich auf falschem Branch war (feature/agentic-fix statt main). Pre-commit-Check: `git branch --show-current` in jeden Commit-Workflow integrieren.
2. **MSYS_NO_PATHCONV** nötig für `gh api /repos/...` Calls auf Git Bash Windows — sonst werden Pfade zu C:/Program Files/Git/repos/... rewritten.
3. **Private Submodules + GITHUB_TOKEN = klonen unmöglich** (scope limitation). Optionen: Repo public, PAT secret, oder Vendor-Copy.
