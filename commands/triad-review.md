# /triad-review — Adversarial Security & Stability Review

Start eine Triad Review mit 3 spezialisierten Angreifern.

**Ablauf:**
1. Target-Dateien identifizieren (unchanged → skip)
2. 3 Angreifer parallel starten (Breaker, Sneak, Scalpel)
3. PoCs statisch validieren
4. Cross-Validation durchführen
5. Report mit Priorisierung erstellen
6. Auf Fix warten → Re-Validierung (ohne neue Agents)
