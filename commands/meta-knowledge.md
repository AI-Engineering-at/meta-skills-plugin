description: "Knowledge Funnel — log errors/learnings, search knowledge, sync to open-notebook, audit consistency"
---

Manage the unified knowledge system. Modes:

- `/meta-skills:knowledge log` — Document a new error (E-number) or learning (L-number)
- `/meta-skills:knowledge search <query>` — Search across open-notebook, ERRORS.md, LEARNINGS.md, Honcho
- `/meta-skills:knowledge sync` — Sync ERRORS.md + LEARNINGS.md to open-notebook as Sources
- `/meta-skills:knowledge audit` — Check consistency: E/L numbers, checkliste references, last sync date
