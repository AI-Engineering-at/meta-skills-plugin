---

Invoke the meta-skills:creator skill to start the cooperative skill creation process.

This is the recommended way to create new skills. It will:
1. Check if the skill already exists (duplicate detection)
2. Position it in your skill ecosystem
3. Build it cooperatively with you (one question at a time)
4. Optimize it for token efficiency (6-step optimization pass)
5. Give you feedback on what you learned
